Resumen de Ejecución
VP Exitosos
Los puntos de verificación se utilizan para comprobar la realización de una acción o verificar el estado o propiedad de un objeto en específico. Al momento de crear un punto de verificación se debe ingresar la información base para realizar la comparación con la información obtenida durante la ejecución. Un punto de verificación exitoso se obtiene cuando se realiza la comparación y ambas informaciones son coincidentes.

VP Rechazados
Los puntos de verificación se utilizan para comprobar la realización de una acción o verificar el estado o propiedad de un objeto en específico. Al momento de crear un punto de verificación se debe ingresar la información base para realizar la comparación con la información obtenida durante la ejecución. Un punto de verificación rechazado se obtiene cuando se realiza la comparación y se presentan variaciones entre la información base y la obtenida en ejecución.
Alertas
Las alertas corresponden a advertencias del programa, las que pueden indicar fallas de ejecución, excepciones del sistema o pasos del proceso con algún comportamiento no adecuado, lo que podria influir a que una ejecución posterior no se concrete de forma exitosa.
